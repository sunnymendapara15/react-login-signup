# React Login Signup

This project showcases a minimal login/signup UI built with React and Vite. It keeps all logic on the client, so you can easily integrate it with your own authentication backend later.

## Getting Started

```bash
npm install
npm run dev
```

Then open the URL provided by Vite (usually http://localhost:5173).

## Features

- Toggle between login and signup modes.
- Simple stateful form handling.
- Instant feedback message on submit.
- Styled components using plain CSS.

Feel free to hook it up to an API or authentication provider of your choice.
